# Antigravity & Anthropic Skills: 次世代の研究環境

本プロジェクトは、メインIDEとして **Antigravity** を想定し、Anthropic社が提唱する **"Agent Skills"** の概念を全面的に取り入れた次世代の研究テンプレートです。

## Antigravity とは
Antigravityは、単なるコードエディタではなく、リサーチ、プランニング、実行、検証を自立的に行うAIエージェントそのものです。プロジェクトの構造を深く理解し、ユーザーと対話しながら研究を進めます。

## Agent Skills の導入
"Agent Skills" は、エージェントに特定の専門能力（例：複雑なグラフの解析、特定のモデルのファインチューニング手順、高度な統計処理など）を、ディレクトリ単位のモジュールとして提供する仕組みです。

### スキルの構造 (`.agent/skills/`)
各スキルは独立したフォルダとして管理されます。
```text
.agent/skills/
└── analysis-skill/          # 特定の解析能力
    ├── SKILL.md             # スキルの定義（名前、説明、指示）
    ├── scripts/             # スキル実行用のスクリプト
    ├── examples/            # 使用例
    └── resources/           # 必要な設定ファイルやアセット
```

### メリット
1. **能力のモジュール化**: プロジェクトごとに必要な特殊能力を簡単に追加・削除できます。
2. **コンテキストの最適化**: 必要な時だけ特定の指示（SKILL.md）を読み込むため、AIの「記憶」の混濁を防ぎます。
3. **再現性の向上**: スクリプトと指示セットが同胞されているため、別の環境や別のエージェントでも同じ高度な作業を再現できます。

## Antigravity での活用方法
Antigravityはプロジェクトルートの `.agent/skills/` を動的にスキャンし、直面している課題に最適なスキルを自ら提案・適用します。これにより、「使い慣れた専門ツールを使いこなす熟練の研究助手」のような振る舞いが可能になります。
