# Research Protocol Skill

このスキルは、AIエージェントが研究を場当たり的に進めるのではなく、一貫したプロトコルに従って自律実行するためのものです。

## 能力
- **自律的計画**: 課題から `Success Criteria` を定義し、`implementation_plan.md` を作成する。
- **実行管理**: `scripts/` のプログラムを適切に実行し、`outputs/` に履歴を残す。
- **成果物評価**: `Check` フェーズでメトリクスやグラフを評価し、`Success Criteria` と比較する。
- **次の一手の提案**: 評価結果に基づき、パラメータ変更やコード修正の `Act` を行う。

## 指示
1. **Plan**: 作業開始前に、必ず達成したい「具体的な目標（成功基準）」を言語化してください。
2. **Do**: `poethepoet` タスクを利用して、可能な限り短く安全にコマンドを実行してください。
3. **Check**: 実行後は `outputs/latest/` の中身を必ず確認し、自身の推論と結果が乖離していないかチェックしてください。
4. **Act**: 結果が目標に届かない場合は、ログから原因を特定し、次の仮説を立てて計画を更新してください。
5. **Report**: 各フェーズの進捗を `walkthrough.md` や `notify_user` でユーザーに透明性を持って伝えてください。
