# データ解析スキル

このスキルは、AIエージェントが実験結果（`outputs/` 配下のCSVやJSON）を読み取り、自動的に統計的な洞察を得るためのものです。

## 能力
- 基本統計量（平均、標準偏差、外れ値等）の算出
- トレンドラインやヒストグラムの描画
- 実験パラメータと結果の相関分析

## 指示
1. **データ特定**: `outputs/latest/` または指定されたディレクトリから主要な成果物（`metrics.json` 等）を特定してください。
2. **解析実行**: `uv run poe edit` を実行して `marimo` 環境でインタラクティブに解析するか、`scripts/analyze_results.py` を用いて自動解析を行ってください。
3. **報告**: 解析結果や可視化（marimoのエクスポート含む）を `report.md` または `report.html` として保存し、主要な発見をユーザーに報告してください。

## 推奨ライブラリ
- `pandas`: 表データ操作
- `matplotlib`/`seaborn`: 可視化
- `scipy`: 統計検定
